# GIK339 - Tutorial
Detta repository innehåller den kod som jag publicerar i en videoserie för kursen GIK339 på Högskolan Dalarna. 
Tutorial och repo är uppsatt så att det finns en branch för respektive video. Vissa delar har startkod och då finns en sådan mapp tillhandahållen från början i den givna branchen. 
Koden du ser i repositoryt är det som jag producerat under videornas gång. Du behöver alltså skapa en egen lagring på din dator eller i ett eget repo om du vill följa med och skriva egen kod. 

Koden i detta repo ska ses som ett stöd för ditt genomförande av tutorial och projekt - inte något att kopiera rakt av (jag kommer att märka det och ni kommer att underkännas om jag ser helt kopierad kod i era inlämningar.)
